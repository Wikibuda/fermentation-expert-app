# Integración con Shopify

## Pasos para implementar:

1. **Copiar contenido** de `shopify_iframe_page.html`
2. **Crear nueva página** en Shopify Admin
3. **Pegar código HTML** en editor de páginas
4. **Añadir al menú** de navegación principal
5. **URL resultante**: /pages/asistente-fermentacion-avanzado

## Características implementadas:

✅ Detección automática de iframe
✅ Ajuste dinámico de altura
✅ Comunicación con contenedor padre
✅ Manejo de errores
✅ Diseño responsive
✅ CTA a productos relacionados
